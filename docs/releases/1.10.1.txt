=========================
Omnileads 1.10.1 release notes
=========================

This is a "bugfix" release improving the stability and performance of the codebase.


Fixes and improvements
=========================
- Fixed error when agent filtered recordings of calls engaged dispositioned
- Fixed inconsistency on campaign inbound report, now all data shown is bounded to the date filter chosen by the user
- Fixed error in supervision inbound campaigns report if the supervisor is not administrator.
- Fixed error that overwrites client custom SSL certificates when upgrading instance
- Added new options to RtpEngine component
- Asterisk component was configured to avoid the error 'Too much open files'
